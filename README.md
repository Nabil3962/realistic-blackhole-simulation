# 🌌 Interstellar Black Hole Simulation

A cinematic, realistic black hole simulation built using HTML5 Canvas and JavaScript.

## Features
- Event Horizon
- Photon Ring
- Accretion Disk
- Space Particles
- Cinematic Glow
- Interstellar Style Animation
